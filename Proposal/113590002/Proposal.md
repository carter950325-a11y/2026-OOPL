# Game Introduction

《PICO PARK: Classic Edition》是一款風靡的 2D 合作解謎動作遊戲（Steam 可免費下載）
主打「團隊合作」與「歡樂友情破壞」。遊戲支援 2 至 10 人在本地或線上連線遊玩，
玩家需扮演彩色方塊貓，透過靈活配合來解開 20 個經典關卡，核心功能包含:
1. 角色控制(移動、跳躍、射擊)
2. 關卡特性設計
3. 地圖碰撞判定
4. 角色之間的互動

## 專案目標
還原最低限度為五個關卡的遊戲內容
1. 推箱子+拿鑰匙開門 (第一關)
2. 角色堆疊跳躍+按按鈕+復活機制 (第二關)
3. 多玩家協同控制一角色+怪物機制+記錄點+範圍移動地圖物件 (第四關)
4. 子彈+有條件消失的地圖物件 (第五關)
5. 玩家間繩索 (第十三關)
