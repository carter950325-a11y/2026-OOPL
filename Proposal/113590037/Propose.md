* **組員：** 113590037 黃閔陽

### Game Introduction
這是一款經典的街機射擊遊戲。玩家在畫面下方控制一艘太空船，可左右移動並發射子彈，對抗上方不斷左右移動並逐漸逼近的外星人艦隊。遊戲包含不同種類的敵人（具備不同行為或分數）、玩家防護罩（會被子彈破壞）等機制。目標是清空所有敵人，若敵人抵達底部或玩家失去所有生命值則遊戲結束。此專案將大量運用 OOP 的繼承與多型概念來建構不同類型的實體。

### Development timeline
* **Week 2：準備工作與素材找尋**
  * 重新撰寫 Proposal 並尋找太空船、外星人、子彈與防護罩圖片。
* **Week 3：環境搭建與基礎渲染**
  * 視窗初始化，讓玩家戰機可以在畫面底部顯示並左右移動。
* **Week 4：射擊系統**
  * 實作玩家發射子彈的機制與子彈向上移動的物理邏輯。
* **Week 5：敵人陣列生成與移動**
  * 使用二維陣列或 List 生成外星人艦隊，並實作群體左右移動與碰壁下移的邏輯。
* **Week 6：碰撞偵測與消除**
  * 實作子彈與敵人的 Hitbox 偵測，擊中時敵人消失並增加分數。
* **Week 7：期中整合與初步優化**
  * 整合玩家移動、射擊與敵人受擊邏輯，確保核心玩法運作。
* **Week 8：期中緩衝週**
  * 補強進度或重構程式碼，確保繼承架構正確。
* **Week 9：敵人攻擊與生命機制**
  * 實作敵人隨機發射子彈，以及玩家中彈扣除生命值（Lives）的系統。
* **Week 10：防護罩系統**
  * 實作位於玩家上方、可抵擋雙方子彈的防護罩，承受一定傷害後會破損。
* **Week 11：UI 介面與遊戲狀態**
  * 製作主選單、分數看板，以及 Start / Game Over 狀態切換。
* **Week 12：音效與視覺特效**
  * 加入射擊、爆炸音效，以及敵人死亡時的圖片切換特效。
* **Week 13：初步測試與記憶體管理**
  * 確保沒有 Memory Leak，子彈飛出邊界或擊中目標後有被正確回收。
* **Week 14：Debug 與邊界測試**
  * 修復各種極端操作下的邏輯錯誤。
* **Week 15：試玩與難度微調**
  * 調整敵人開火頻率、子彈速度與分數平衡。
* **Week 16：最後檢查**
  * 程式碼重構，確保符合封裝、繼承、多型等 OOP 原則。
* **Week 17：交件準備**
  * 錄製示範影片、製作簡報與文件。
