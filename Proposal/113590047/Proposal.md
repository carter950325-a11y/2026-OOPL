# Game Introduction

植物大戰殭屍是一款 2D 類型塔防遊戲，玩家必須要擊倒殭屍保護最後防線後進入下一關，製造植物需要能源。  
在打每關之後會獲得不同種類的植物，殭屍也會出現不同種類，玩家必須要了解不同植物在地圖上的優勢，才能發揮最大的效果。  

[遊戲連結](https://www.youtube.com/watch?v=XENla8M3910&t=821s)
